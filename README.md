SMS Spam Detection Dataset

A machine learning dataset for SMS spam classification and text analysis.
This dataset contains labeled SMS messages categorized as spam or ham (non-spam) and can be used for Natural Language Processing (NLP) and machine learning projects.

📌 Overview

The dataset includes SMS messages with two columns:

Column	Description
v1	Label (spam or ham)
v2	SMS message text

This dataset is commonly used for:

Spam message classification
NLP preprocessing
Text mining
Machine learning model training
Deep learning experiments
